## 1. HTML
1. html은 기본적으로 태그 언어
2. 웹사이트의 뼈대를 구성하는 태그들

```html
<html>
    <head>
    </head>
    <body>
    </body>
</html>
```

3. SPA(single page application)
- 하나의 페이지만 존재하는 웹 어플리케이션
- spa는 하나의 페이지로, html은 1개로, content만 동적으로 body 태그 안에 넣는 형태임
- 주로 react가 여기서 적용되는 것으로 생각하면 됨

4. css (cascading style sheets)
- 웹 페이지의 디자인을 담당하는 부분
- html 소스는 동일하지만 css에 따라 다양한 디자인으로 변경 가능함


## 3. Javascript
- parameters
- arguments
    
```javascript
//function statement
function sum(a,b) {
    return a+b;
}

console.log(sum(10,20));
//출력 결과 : 30

//arrow function expression
const multiply = (a,b) => {
    return a * b;
}
console.log (multiply(10,20));
//출력 결과 200
```
