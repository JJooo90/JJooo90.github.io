## 1. 글자 쓰기 관련
### 1. 하이라이트
  글에 형광팬같은 하이라이트를 부여하고 싶을 때 \`를 붙여주면 된다 
  > 작성 예시 : \`test\`  
  >> `test`

### 2. 마크다운 문법을 글에 표현할 때
\ 를 앞에 붙여주면 됨 (Back slash 입력)
> 작성 예시 : \\&nbsp  
>> \&nbsp

## 2. 문장 작성 관련
### 1. 띄어쓰기 
특수코드 \&nbsp; 를 입력 하거나,<br>
전각문자 띄어쓰기(공백) 을 입력하는 방법이있다.<br>
전각 공백: \ <br>
위 검은 공간을 복붙 하면된다.<br>


> ├── 📁_data           # data files for customizing the theme<br>
  ├── 📘navigation.yml    &nbsp;# main navigation links<br>
  └── 📘ui-text.yml      # text used throughout the theme's UI<br>
 

<br>

### 2. 줄 바꿈
\<br/> 태그 원하는 위치에 사용하거나, 공백문자를 문장끝에 두번 사용하면 줄바꿈이 가능하다.
> 작성 예시 : 요건 테스트 \<br> 하는 겁니다용 <br>
>> 요건 테스트 <br> 하는 겁니다용

### 3. 코드 블록
\`\`\` 쓰고싶은 말 \`\`\`을 작성하면 됨
> 작성 예시 : \`\`\` 테스트 합니다요 \`\`\`

``` 
쓰고싶은 말 
`git`  '쓰는' 중입니다. 
```
